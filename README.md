# Mental Arithmetic Trainer
The application is inspired by zetamac, designed to be playeed in zen infinite mode and through offline. A training tool to enhance mental mathematics if your goal is to solve 100 aritmetic questions in 10 minute. Are you up for the challenge?

### Tip:
set minimum to 11 and maximum to 99 for better results. This is the link: https://cymondave.github.io/Arithmetic-Trajner/
